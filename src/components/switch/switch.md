用于在打开和关闭状态之间进行切换。

## API

| 属性            | 说明                  | 类型                                                                       | 默认值    | 版本 |
| ------------    | -------------------- | -------------------------------------------------------------------------- | --------- | ---- |
| checked         | 指定当前是否选中       | boolean                                                                     | `false` |      |
| defaultChecked  | 初始是否选中          | boolean                                                                     | `false`   |      |
| color        | 选中后颜色              | string                                                                     | `green`  |      |
| onChange        | 变化时回调函数        | (value:boolean,event:React.MouseEvent<HTMLElement>) => void                  |           |      |
