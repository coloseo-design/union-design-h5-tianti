## API

| 属性            | 说明                  | 类型                                                                       | 默认值    | 版本 |
| ------------    | -------------------- | -------------------------------------------------------------------------- | --------- | ---- |
| color         | 弹出框颜色       | `white` , `red`, `black`                                                           | `white` |      |
| content        | 弹出框内容          | React.ReactNode                                                                     |           |      |
| placement        | 弹出框位置              |   `top` , `topLeft`, `topRight`, `bottom`,`bottomLeft`,`bootomRight`          | `top`  |      |
