## Api

| 参数             | 说明                   | 类型              | 默认值   |
| :--------------- | :--------------------- | :---------------- | :------- |
| round         | 圆形标签            | boolean | - |
| style            | 样式                   | CSSProperties     | -        |
| className        | 样式                   | string            | -        |
| hollow     | 空洞标签     | boolean     | -        |
| big | 大标签     | boolean            | -        |
| onClick        | 点击标签触发 | React.MouseEventHandler<HTMLElement>     | -        |
