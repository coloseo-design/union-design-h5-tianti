## API

| 参数      | 说明           | 类型          | 默认值 |
| :-------- | :------------- | :------------ | :----- |
| visible   | 是否显示遮罩  | boolean       | false  |
| style     | 样式           | CSSProperties | -      |
| className | 样式           | string        | -      |
| onClick   | 点击 遮罩 事件 | ()=>void      | -      |
