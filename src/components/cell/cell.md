## API

| 参数      | 说明           | 类型          | 默认值 |
| :-------- | :------------- | :------------ | :----- |
| style     | 样式            | CSSProperties | -      |
| className | 样式            | string        | -      |
| subtitle  | 小标题          | string \| React.RactNode        | -      |
| title     | 标题            | string \| React.RactNode        | -      |
| content   | 内容            | string \| React.RactNode     | -      |
| footer    | 底部内容        | string \| React.RactNode  | -  |
