在页面中间弹出黑色半透明提示，用于消息通知、加载提示、操作结果提示等场景。

## API

## Toast

| 属性 | 说明 | 类型 | 默认值 | 版本 |
| --- | --- | --- | --- | --- |
| content | 提示内容 | React.Element or String | - |  |
| duration | 自动关闭的延时，单位秒 | number | 3 |  |
| onClose | 关闭后回调 | Function | - |  |
| mask | 是否显示透明蒙层，防止触摸穿透 | Boolean | true |  |
